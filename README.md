# 有給休暇管理システム

医療法人向けの有給・特別休暇管理アプリ。ブラウザ単体（ビルド不要）で動作し、バックエンドは Supabase。

## 構成

- `index.html` … アプリ本体（HTML / CSS / JavaScript 一体）
- `schema.sql` … Supabase(PostgreSQL) のスキーマ（適用済みの記録用）

## ローカルでの起動

```sh
python3 -m http.server 8080
```

→ ブラウザで http://localhost:8080 を開く。
（同じ Wi-Fi のスマホからは `http://<MacのIP>:8080`）

## 主な機能

- カレンダー中心のUI（全スタッフの休暇予定を共有表示・日本の祝日表示）
- 有給／誕生日休／代休の取得・取消（承認フローなし＝申請即確定）
- 1日・半日・時間単位（1時間〜）の取得
- 法定付与の自動計算（入職日基準・カットオフ 2026-07-03 以降）／誕生日休の自動付与
- 管理者：スタッフ管理、付与・取得の代理登録／削除、月次レポート（CSV）
- 権限：管理者 / スタッフ / 閲覧専用

## バックエンド

- Supabase（PostgREST + GoTrue Auth + Row Level Security）
- **スタッフ情報・付与/取得などの実データは Supabase 側に保存**されており、このリポジトリには含まれません（コードのみ）。

## メモ

- 単一HTMLのため、`index.html` を編集して保存 → ブラウザ再読み込みで反映（ビルド不要）。
